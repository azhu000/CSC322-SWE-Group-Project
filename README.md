# CSC322-SWE-Group-Project
Group project for CSC33200

Initial setup:
- install python (3.9)
- install flask via pip: `pip install Flask`
- run flask :
- ```bash
    set FLASK_APP=/full/path/to/your/folder/app.py
    python -m flask run
  ```
- Open your browser to : `http://localhost:5000/?name=JoeShmoe`
